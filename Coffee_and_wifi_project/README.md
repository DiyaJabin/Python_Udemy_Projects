\# Day 62 – Cafe \& Wifi Website (Flask + WTForms + Bootstrap)



Course project from a Udemy Python course.



This project is a Flask-based web application that allows users to view and add cafes with information about wifi, power availability, and coffee quality.



The main focus of this project is working with \*\*Flask-WTF forms\*\*, \*\*Bootstrap-Flask\*\*, \*\*Jinja template inheritance\*\*, and basic \*\*CSV data handling\*\*.



\## Features

\- View a list of cafes with details (location, timings, ratings)

\- Add a new cafe using a form

\- Form validation using WTForms

\- Template inheritance for reusable layouts

\- Data stored and updated in a CSV file

\- Styled using Bootstrap



\## Tech Used

\- Python

\- Flask

\- Flask-WTF (WTForms)

\- Bootstrap-Flask

\- Jinja2 (Template Inheritance)

\- HTML/CSS

\- CSV



\## Run

1\. Install dependencies:

&#x20;  pip install flask flask-wtf bootstrap-flask

2\. Run the app:

&#x20;  python main.py

3\. Open in browser:

&#x20;  http://127.0.0.1:5000
