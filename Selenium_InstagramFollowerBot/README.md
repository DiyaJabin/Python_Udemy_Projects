\# Day 52 – Instagram Follower Bot



Course project from a Udemy Python course.



Uses Selenium WebDriver to:

\- Log into Instagram

\- Open a target account’s followers list

\- Scroll the followers popup to load more followers (automation practice)



Credentials and target account are read from a `.env` file (not committed).



\## Status

Paused for now. Instagram’s UI and element selectors change frequently, which can break automation scripts.

I’ll revisit this later to improve reliability.
