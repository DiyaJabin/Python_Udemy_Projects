\# Dr. Semmelweis Handwashing Discovery Analysis



\## Overview



This project explores the historical medical data collected by Dr. Ignaz Semmelweis between 1841 and 1849 to investigate the impact of mandatory handwashing on maternal mortality rates.



Using Python, Pandas, Matplotlib, Seaborn, Plotly, and SciPy, I analyzed birth and death records from Vienna General Hospital and visualized how handwashing significantly reduced deaths caused by childbed fever.



\## Technologies Used



\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn

\- Plotly

\- SciPy

\- Jupyter Notebook



\## Key Analysis



\- Data exploration and cleaning

\- Comparison of hospital clinics

\- Handwashing impact analysis

\- Rolling average trend analysis

\- Box plots and histograms

\- Kernel Density Estimation (KDE)

\- Statistical hypothesis testing using t-tests



\## What I Learned

\## Key Finding



The analysis showed that introducing mandatory handwashing reduced maternal mortality rates dramatically, and statistical testing confirmed that the reduction was highly significant rather than due to chance.
