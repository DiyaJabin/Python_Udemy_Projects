\# Day 57 – Flask Blog with Jinja Templating



Course project from a Udemy Python course.



This project demonstrates how to use \*\*Jinja templating in Flask\*\* to dynamically render content on a webpage.



Features:

\- Flask backend serving HTML templates

\- Blog posts loaded from an external API

\- Dynamic page generation using Jinja loops and conditionals

\- Individual blog post pages rendered based on the post ID



\## Tech Used

\- Python

\- Flask

\- Jinja2

\- HTML/CSS



\## Run

1\. Install dependencies:

&nbsp;  pip install flask requests

2\. Start the server:

&nbsp;  python server.py

3\. Open in browser:

&nbsp;  http://127.0.0.1:5000
