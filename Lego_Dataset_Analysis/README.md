\# Day 74 - LEGO Dataset Analysis



This project is part of my Python learning journey where I explored and analyzed LEGO datasets using Python, Pandas, and Matplotlib in a Jupyter/Google Colab notebook.



The project focuses on understanding LEGO themes, sets, colours, trends over time, and data relationships through exploratory data analysis and visualization.



\---



\## Topics Covered



\- Data exploration using Pandas

\- GroupBy and aggregation operations

\- Data visualization using Matplotlib

\- Merging DataFrames

\- Trend analysis on LEGO datasets

\- Working with CSV datasets

\- Basic data preprocessing



\---



\## Questions Explored



\- How many unique LEGO colours exist?

\- How many transparent vs opaque colours are there?

\- What were the first LEGO sets released?

\- Which LEGO sets had the highest number of parts?

\- How did LEGO set releases change over time?

\- Did LEGO sets become larger and more complex over the years?

\- Which LEGO themes contain the highest number of sets?
