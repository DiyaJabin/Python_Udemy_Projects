\# Nobel Prize Analysis



\## Overview



This project explores Nobel Prize data from 1901–2020 using Python data analysis and visualization libraries. The dataset was cleaned, analyzed, and visualized to uncover trends related to Nobel Prize categories, countries, organizations, gender distribution, and laureate demographics.



\## Technologies Used



\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn

\- Plotly

\- Jupyter Notebook



\## Key Analysis



\- Data cleaning and preprocessing

\- Nobel Prize trends over time

\- Gender distribution of laureates

\- Country-wise Nobel Prize analysis

\- Top organizations and institutions

\- Laureate age distribution

\- Interactive visualizations with Plotly



\## What I Learned



\- Exploratory Data Analysis (EDA)

\- Data cleaning with Pandas

\- Data visualization techniques

\- Working with real-world datasets

\- Extracting insights from historical data
