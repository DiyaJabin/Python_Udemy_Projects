\# Day 75 – Google Trends Data Analysis



Course project from a Udemy Python course.



This project explores the relationship between Google search trends and real-world data such as Tesla stock price, Bitcoin price, and unemployment rates.



The analysis is done using Python, Pandas, and Matplotlib in a Jupyter/Google Colab notebook.



\## Topics Covered

\- Data exploration with Pandas

\- Checking and handling missing values

\- Converting strings to datetime objects

\- Resampling daily data into monthly data

\- Rolling averages

\- Time series visualization

\- Dual-axis line charts using Matplotlib



\## Questions Explored

\- Does Tesla search popularity relate to Tesla stock price?

\- Does Bitcoin news search volume relate to Bitcoin price?

\- Can searches for “Unemployment Benefits” show patterns related to the actual unemployment rate?

\- How does including 2020 affect unemployment search trend analysis?



\## Data Sources

\- Google Trends

\- Yahoo Finance

\- FRED unemployment data



\## Tools Used

\- Python

\- Pandas

\- Matplotlib

\- Jupyter Notebook / Google Colab



\## Project Type

Course-guided exploratory data analysis project focused on time series analysis and visualization.
