\# Day 58 – TinDog Website (Flask + Bootstrap)



Course project from a Udemy Python course.



This project is a responsive landing page built using \*\*Flask\*\* and \*\*Bootstrap CSS\*\*.



Features:

\- Flask backend serving HTML templates

\- Responsive UI using Bootstrap

\- Multiple sections (features, testimonials, pricing)

\- Clean and modern landing page design



\## Tech Used

\- Python

\- Flask

\- Bootstrap

\- HTML/CSS



\## Run

1\. Install Flask:

&#x20;  pip install flask

2\. Start the server:

&#x20;  python server.py

3\. Open in browser:

&#x20;  http://127.0.0.1:5000
