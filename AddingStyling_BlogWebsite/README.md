\# Day 59 – Anime Blog Website (Flask + Jinja + Bootstrap)



Course project from a Udemy Python course.



This project is a simple blog website built using \*\*Flask\*\*, \*\*Jinja templating\*\*, and \*\*Bootstrap\*\*.



Features:

\- Flask backend with routing

\- Dynamic content rendering using Jinja templates

\- Responsive UI built with Bootstrap

\- Multiple pages (home and individual post views)

\- Clean and modern blog-style layout



\## Tech Used

\- Python

\- Flask

\- Jinja2

\- Bootstrap

\- HTML/CSS



\## Run

1\. Install dependencies:

&#x20;  pip install flask

2\. Start the server:

&#x20;  python server.py

3\. Open in browser:

&#x20;  http://127.0.0.1:5000
