# Day 63 – Virtual Library App (Flask + SQLite + SQLAlchemy)

Course project from a Udemy Python course.

This project is a simple Flask web application for managing a personal book library.

The main focus of this project is learning how to work with **SQLite databases** using **SQLAlchemy ORM** in a Flask application.

## Features
- View all books in the library
- Add a new book with title, author, and rating
- Edit an existing book rating
- Delete a book from the library
- Store data in a SQLite database
- Use SQLAlchemy ORM models for database operations

## Tech Used
- Python
- Flask
- SQLite
- SQLAlchemy
- Flask-SQLAlchemy
- HTML
- Jinja2

## Run
1. Install dependencies:
   pip install flask flask-sqlalchemy
2. Run the app:
   python main.py
3. Open in browser:
   http://127.0.0.1:5000
