\# Day 64 – Top 10 Anime Website



Course project from a Udemy Python course.



This project is a Flask web application for creating and managing a personal Top 10 anime list.



It combines Flask, SQLite, SQLAlchemy, WTForms, Bootstrap, and API requests to build a database-driven web app.



\## Features

\- View anime entries in a ranked list

\- Add a new anime by searching through an external anime API

\- Store anime details in a SQLite database

\- Update rating and review using WTForms

\- Delete anime entries

\- Display anime images, descriptions, ratings, and reviews



\## Tech Used

\- Python

\- Flask

\- Flask-Bootstrap

\- Flask-WTF / WTForms

\- SQLite

\- SQLAlchemy

\- Jinja2

\- Requests

\- HTML/CSS



\## Run

1\. Install dependencies:

&#x20;  pip install flask flask-bootstrap flask-wtf flask-sqlalchemy requests

2\. Run the app:

&#x20;  python main.py

3\. Open in browser:

&#x20;  http://127.0.0.1:5000
